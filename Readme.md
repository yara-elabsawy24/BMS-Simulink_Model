# BMS Simulink Model

This repository contains a Simulink model developed for a Battery Management System (BMS) thesis project.

## Requirements
- MATLAB R2025b 
- Simulink
- Embedded Coder
- STM32 Support Package

## Files
- `BMS_Model.slx` : Main Simulink model
- `parameters.m` : Model parameters
- `battery_data.mat` : Battery data

## How to Run
1. Open MATLAB.
2. Run `parameters.m`.
3. Open `BMS_Model.slx`.
4. Start the simulation.
